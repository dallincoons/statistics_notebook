1 Quantitative Variable

Quantatative variables are measured on a numeric scale. An example would be like height or weight. In addition to being numeric, it should make sense to perform arithmetic on this type of data. For example, a phone number is numeric but it wouldn’t make sense to take the mean of a list of phone numbers, so that’s an inidication you’re not dealing with quantative data.

A histogram or barchart is a good visualization tool for this type of data.

Good numerical summaries to use are median, five-number summary

Mean and standard deviation are great if this is normally distrubuted.

Examples of tests that are a good match for this type of data are One Sample t Test and Signed-Rank Wilcoxon Test

1 Quantitative Variable | 2 Groups

Y is the data that we’re interested in learning more about

X is the explanatory varable which gives greater context to Y

For example, if we want to study how many chocolate bars people ate in a year according to gender, the number of chocolate bars would be the numeric variable we’re interested in, and the gender is the category or the variable that gives us more context. In this case we might run an Independent samples t-test where one group is number of chocolate bars eaten by males, and the other group is the number of bars eaten by females.

Scatterplots or dot plots could be good for this type of data.

Examaple of a test you would use with this type of data is Independent samples t-test or Wilcoxon Rank Sum

One of the most common tests with this type of data is the t-test, but it requires normal data or a large sample size, so if sample size is small and not normal, or if you’re working with ordered, quantatative data, a Wilcoxon Rank Sum test is an option to consider. A Signed-Rank Wilcoxon Test could also be used for paired data.

An example of a barplot would be something like:

boxplot(length ~ sex, data=KidsFeet)

1 Quantitative Variable | 3+ Groups

Like 2 group Quantitative Variable but with more groups. For example, if we wanted to test to see if there is a day of the week that is more busy than another day, we could collect the number of visits for each day of the week, which would give us 7 groups of quantatative variables. We could then run something like an Anova test to see if one of the days tends to have a different number of visits than the others.

Box plot is good for this type of data.

Example of tests to use are Anova or Kruskal-Wallis Rank Sum

A Kruskal-Wallis Rank Sum is a good option to consider for ordered quantatative data and/or data that is not normal. Otherwise, for normal, qualatative data, Anova would be appropriate.

2 Quantitative Variables

With this type of data, often we’re interested in discovering which quantatative variables have an effect on another quantatative variable. For example, if you wanted to find out how much taller a person is likely to be based on their shoe size, height and shoe size could be X and Y. A test that can be used in this case could be a Single Linear Regression.

A visual example of this type of data would be human feet measured by length and width.

plot(length ~ width, data=KidsFeet)

pander(KidsFeet %>% summarise(cor(width, length)))
cor(width, length)

1 Quantitative Response | Multiple Explanatory Variables

This would involve a response variable with 2 or more explanatory variables. Building on the example from the 2 Quantative Variables section, this could involve including weight as well as gender to predict a persons height, or perhaps we want to predict the price of a car given it’s model and mileage. Multiple Regression is a test that can be used with this type of data.

Binomial Response | 1 Explanatory Variable

X is the explanatory variable, Y is the response, so that we’re predicting where Y will be in relation to 1 or 0 depending on where X is. For example, if we want to predict whether an O-ring will fail or not based on the temperature, this would be a binary response - either it fails or it doesn’t.

Another example of binomial response is determining whether someone is above or below a certain age. Although normally age is quantative, we can turn almost anything into a binomial response. In this case age would be the Y variable and X might be another attribute like height.

An example of a test that works with binomial response using multiple explanatory variables is a logistic regression

Binomial Response | Multiple Explanatory Variables

We might have multiple X explanatory variables that are useful in predicting the response variable Y.

For example if we want to predict whether a person is above or below a certain age (response variable, Y), we could use multiple X explanatory variables such as height, or facial hair.

An example of a test that works with binomial response using multiple explanatory variables is a multiple logistic regression
